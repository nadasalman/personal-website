Module taught with practical sessions, hackathon and group projects (30 hours total). Topics covered:

- Introduction to Machine Learning and model development lifecycle
- Data preparation and exploratory data analysis
- Supervised learning: classification and regression
- Model evaluation and performance metrics
- Feature selection and dimensionality reduction
- Hyperparameter optimisation
- Ensemble learning methods (Random Forest, Gradient Boosting)
- Unsupervised learning and clustering
