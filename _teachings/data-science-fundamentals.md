Module designed and taught (25 hours total). Topics covered:

- MATLAB programming: variables, matrices, control structures, functions and data visualisation
- Code debugging and analysis
- Interpolation, polynomial regression and error analysis
- Numerical solution of non-linear equations and linear systems
- Numerical methods for ODEs: Euler, Runge-Kutta, ode45
- Modelling and simulation with Simulink
